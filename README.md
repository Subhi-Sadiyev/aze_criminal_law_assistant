**Notes on reference patterns**: 
56.1.1—56.1.5 means 56.1.1, 56.1.2, 56.1.3, 56.1.4 and 56.1.5.
"72–73-2-ci maddələrində" means 72.2 and 73.2 i.e second MaddəSection of Maddə 72 and 73. This means hyphen does not always denote range.
Some patterns like "Bu Məcəllənin 192,  193, 195, 195-1, 196–199, 202-2, 203, 203-1.1, 205-2,  210–212 və 213-2-ci maddələrində" include extra space unlike others. This pattern also includes both usage of hyphen as range in "210-212" and as number of direct MaddəSection in "213-2-ci". This means you need several regex patterns to go through and unpack both ranges and direct references. 
In the following pattern "bu Məcəllənin 99-4.1.1—99-4.1.3-cü maddələrində" means 99-4.1.1 through 99-4.1.3. This means some MaddəSections have sub-sections themselves.
In the following long pattern "bu Məcəllənin 104-1, 137, 144, 144-1, 144-2, 144-3, 167–169, 169-1, 170–171-2, 192-2, 193-1, 194, 200 – 200-3, 206.2-206.4, 206-1, 214–220, 233, 234, 237, 240, 242-244-1, 246, 246-1, 259-261, 271–284, 303.2, 308, 311, 312, 312-1, 313, 315, 316-1, 316-2, 320 və 323–326-cı maddələri" there is a mixture of all previus cases and the special case of "200 – 200-3" means all MaddəSection of Maddə 200 untill 200.3. Additionally "323–326-cı" means MaddəSection 323 through 326.
In the following pattern "Bu Məcəllənin 116.0.8-1-116.0.8-3-cü maddələrində" there are two MaddəSection. 116.0.8-1 and 116.0.8-3.
